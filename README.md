# mini-games

Jueguitos clásicos con **HTML + CSS + JS**.

## Juegos

- Menú (portal tipo FRIV): `./index.html`
- Gato (tatetí): `./gato/`
- Ahorcado: `./ahorcado/`

## Cómo ejecutar

Opción A (simple): abrí el archivo `index.html` con doble click.

Opción B (recomendado): usá un servidor local (por ejemplo, la extensión **Live Server** en VS Code) y abrí la carpeta del repo.
